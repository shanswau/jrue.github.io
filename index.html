Class Syllabus

Sept. 5

Introduction to JavaScript: class will cover syntax, variables, operators, datatypes, concatenation, arrays, multidimensional arrays and displaying information in the console.

Reading: Section on variables in the KDMC online tutorial.

Sept. 12:

Objects: class will cover the three types of object notation: object-literal, dot notation and square-bracket notation; how to store or retrieve data; JSON structures and combining objects with arrays.

Reading: Section on objects in the KDMC online tutorial.

Assignment: Parse a data JSON feed from a popular online site like Twitter, NYT or Instagram.

Sept. 19:

Conditionals: class will cover logical conditionals and comparison operators. We will use jQuery to manipulate the DOM based on a given set of conditions.

Reading: Section on conditionals in KDMC online tutorial.

Sept. 26:

Functions: class will cover how to use functions to execute code blocks, send data to functions via arguments, return data from a function and how to use anonymous functions. Discuss how functions affect variable scope, and how to use encapsulation to avoid cluttering the global scope.

Reading: Section on functions in KDMC tutorial, and begin looking at Code Academy tutorials.

Oct. 3:

The DOM: finally, class will start constructing projects that will manipulate the DOM and put into practice the previous four classes for constructing web projects. We will construct a basic online news quiz in class.

Reading: Section on DOM in KDMC tutorial, jQuery tutorials and Code Academy.

Assignment: Build a news quiz for next class.

Oct. 10:

Loops: class will cover how to iterate through a dataset, like an array or object. We will take a dataset from a popular Web service, and display content depending on the length of the dataset.

Reading: Section on loops in KDMC tutorial.

Assignment: Take tutorial dataset, and display the data visually in the DOM.

Oct. 17:

Workflow: class will cover the process and how to structure some basic programs based on the subjects covered in previous classes. Practical guidelines to keeping code clean, logical and well-commented. Introduction to GitHub for posting future assignments.

Reading: GitHub online tutorial.

Assignment: Post previous assignments to GitHub class account. Study for test following week.

Oct. 24:

APIs: class will cover jQuery and AJAX requests for querying web services for JSON data. Then we will parse the data, and display elements in the DOM. Class will also include an in-class written test on parsing JSON.

Reading: jQuery tutorial on AJAX method.

Assignment: Build an instagram slideshow viewer.

Oct. 31:

HTML5 and document setup: class will cover more on the process and structure of building web projects, including using Modernizr for testing new HTML5 features before implementing them, and setting up a boilerplate template for real-world projects. How to incorporate projects via an iFrame.

Reading: Look at the tutorial templates given by the instructor.

Assignment: Use one of the tutorial templates to construct a web project.

Nov. 7:

Prototyping and OOP: class will cover some of the foundations to prototypical languages and object oriented programming. How to store functions in variables, instantiate objects, object inheritance, polymorphism. How to use libraries which have their own variable scope.

Reading: Read D3 tutorials for next week’s class.

Nov. 14:

Introduction to D3: class will introduce D3 dataviz library for manipulating the DOM.

Reading: Read Jerome Cukier’s D3 tutorial.

Assignment: Recreate one of the charts from the D3 example list.

Nov. 21:

Animation in D3: class will cover basic animation methods, parsing CSV data, and additional D3 methods.

Reading: D3 online tutorials.

Assignment: Create a D3 chart based on a CSV given by the instructor.

Nov. 28:

No Class for Thanksgiving Break

Dec. 5:

Leaflet Maps: class will cover constructing a basic Leaflet map by following their documentation, how to add a marker, and customize the info window.

Reading: Leaflet tutorial.

Assignment: Create a leaflet map based on a JSON dataset given by the instructor.

Dec. 12:

Putting it all together: class will cover how to incorporate a project into a website in a real-world scenario. Using AJAX to query data, loops to iterate through the data, and a third-party library to display the data.
