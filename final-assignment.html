Step 3:

Adding some jQuery for resizing

HTML/CSS is notoriously difficult when it comes to dealing with heights. A page is scrollable, so adjusting heights tends to be a tricky task. JavaScript/jQuery to the rescue!

We can use jQuery to determine the window's height that the user currently has, and then set all of the <section> tags to match.

First, let's add our script tags and our jQuery ready function (don't forget, this is important!)

Next, we want to create a function that triggers when the window loads, but also when it resizes. We can use jQuery's on command, which allows us to trigger events and run a function when those events fire. The nice thing about jQuery is that it allows us to specify multiple events in one declaration.

$(window).on('load resize', function(){

});

This code will execute the anonymous function whenever the window either loads for the first time, or when the user resizes the window.

Next, let's add some code to size all of our <section> tags to the same height as the window, thus creating a panel effect where each section of our project is the same size as the window.

$('section').height( $(window).height() );

This code is essentially saying make the height value of each section the same as the window's height.

Lastly, let's add some CSS which will make each section position relative, so that any absolute elements inside the section will position according to the section.

section{
   position : relative;
}

Step 4:

Centering the box

Let's get the first page looking good. Let's center the page title and tagline. We will use a trick called absolute centering which is a method to set the left, right, top and bottom to zero, so they all compete for eachother. We also set the margins on all sides to auto. To do this, we need to create a wrapping element like a <div> around our h1, h2, and button tags. Give it a memorable id.

Note that absolute centering only works when there is a predefined width and height on the box. Let's do that now in our CSS:

#boxcenter{
   position: absolute;
   text-align:center; 
   top:0;
   right:0;
   bottom:0;
   left:0;
   margin:auto;
   width:150px; /* adapt to your own situation */
   height:150px;
}

Step 5:

Button action

Next, let's make the page scroll down when the user clicks on the button. We will do this with jQuery.

Look at the code in section 3, and see how we used the on command to run a function when an event fired. We're going to use that same process, but this time, we want to perform the action on the 'button' rather than the window. And we don't want to wait for the load or resize events, instead, let's trigger the function using the click event.

So, what do we want to do when the user clicks the button? Let's have the page autoscroll. We can use jQuery's animate function to scroll the page down. There is a special property that jQuery recognizes called "scrollTop" which specifies how much the page needs to scroll. So after the user clicks the button, change the scrollTop property by the window's height, thus giving the illusion that we are scrolling to the next section.

$('html, body').animate({scrollTop: $(window).height()}, 1000);

In order to scroll the page, we need to adjust the scrollTop of both the HTML and body elements.
